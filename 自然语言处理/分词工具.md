## 1.1 安装
pip install spacy  
无法通过 pip 来安装 spacy 的模型，会报网络超时的错误。
因此访问该网站：https://github.com/explosion/spacy-models/releases?q=zh_core_web_sm 找到对应的 .whl 文件
下载 .whl 文件，上传至服务器。然后 pip install path.whl
